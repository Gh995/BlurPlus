# BlurPlus
Blur+ is a highly optimized motion-blur shader for OptiFine 1.8.9




Blur+ Shaderpack
<img src="./logo.png" width="200"/>
A lightweight, optimized shaderpack designed to add smooth blur effects, motion trails, and subtle visual polish to Minecraft 1.8.9 without harming FPS.

Blur+ focuses on performance, stability, and a clean visual style suitable for PvP, building, and general gameplay.
